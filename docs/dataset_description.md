# Dataset Description

## Dataset Source
The dataset used in this project was obtained from Kaggle.

Dataset Link:
https://www.kaggle.com/datasets/amitvkulkarni/impact-of-product-positioning-on-sales

---

## Dataset Overview

The dataset contains information related to product positioning, pricing strategies, promotional activities, consumer demographics, and sales performance.

The objective of this dataset is to analyze how product placement strategies influence sales volume and consumer behavior in retail environments.

---

## Dataset Features

| Column Name | Description |
|-------------|-------------|
| Product_ID | Unique identifier for each product |
| Product_Position | Placement of the product (Endcap, Aisle, Shelf) |
| Price | Selling price of the product |
| Competitor_Price | Price offered by competitors for similar products |
| Promotion | Indicates if the product has promotions or discounts |
| Foot_Traffic | Number of customers passing the product location |
| Consumer_Demographics | Target audience group (Families, Seniors, Young Adults, College Students) |
| Product_Category | Category or type of product |
| Seasonal | Indicates whether the product is seasonal |
| Sales_Volume | Number of units sold |

---

## Purpose of the Dataset

This dataset helps analyze the relationship between product placement strategies and sales performance. It allows data analysts to understand how different store placements and marketing factors influence customer purchasing behavior.
